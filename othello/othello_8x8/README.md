# 8x8 Othello

## 操作教學
安裝keras: 
```shell
pip install keras
```

遊戲介面使用pygame(注意:適用於1.9版本，2.0以上會出現問題)

安裝pygame: 
```shell
pip install pygame==1.9.6rc2
```
[其它版本https://www.lfd.uci.edu/~gohlke/pythonlibs/#pygame](https://www.lfd.uci.edu/~gohlke/pythonlibs/#pygame)

Python版本 = 3.7
Tensorflow版本 < 2.0

## 訓練

python ./train.py
